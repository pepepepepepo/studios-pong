### ⑤ Goton Weight（語温加重ノルム）実装 ✅

`core/resonance/resonance_engine.py`:

```python
# StructureVector に追加
goton_weights: Optional[Tuple[float, float, float, float]] = None

def weighted_norm(self) -> float:
    """‖S_i‖_w = √(Σ wᵢ·xᵢ²)  — 語温加重ノルム"""
    if self.goton_weights is None:
        return self.norm()
    wt, wd, wi, wc = self.goton_weights
    return math.sqrt(wt*T² + wd*D² + wi*I² + wc*C²)
```

```python
engine.register("shizuku", goton_weights=(3.0, 1.0, 0.5, 1.0))
# タグ次元に3倍感敏感、干渉はやや鈍感なしずく
```
